Kafka Offset Monitor Setup

發表於 2016-03-27   |  

How to fire up a kafka offset monitor instance:

  1. Download jar from github: https://github.com/quantifind/KafkaOffsetMonitor/releases
  2. Fire the java process at port 7088:
    • java -cp KafkaOffsetMonitor-assembly-0.2.1.jar com.quantifind.kafka.offsetapp.OffsetGetterWeb –zk ebs-ali-beijing-mig:2181 –port 7088 –refresh 10.seconds –retain 2.days &
  3. Open your browser and access : http://localhost:7088
