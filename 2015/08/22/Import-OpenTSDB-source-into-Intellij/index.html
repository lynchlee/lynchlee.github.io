Import OpenTSDB source into Intellij

發表於 2015-08-22   |   分類於 TSDB   |  

Env:

  1. Clone opentsdb from github $git clone git@github.com:OpenTSDB/opentsdb.git

  2. build $cd opentsdb && ./build.sh

  3. create dev folder $mkdir opentsdb-dev

  4. now you have two dirs name opentsdb and opentsdb-dev

  5. $cd opentsdb-dev && ln -s /work/opentsdb/build/third_party libs

  6. $mkdir -p src/main/java/net && cd src/main/java/net

  7. $ln -s /work/opentsdb/src opentsdb

  8. $mkdir -p src/main/java/tsd && ce src/main/java/tsd

  9. $ln -s /work/opentsdb/src/tsd/client client

  10. $mkdir -p src/test/java/net && cd src/test/java/net

  11. $ln -s /work/opentsdb/test opentsdb

  12. Fire up your Intellij , import the source into , and then enjoy !!

OK, done !!
